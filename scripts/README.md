
# Scripts

The scripts folder contains all the scripts necessary to fully reproduce
the benchmarking manuscript in chronologically ordered subfolders. Each
subfolder contains a readme file with further explanations of the
different scripts and what they
do:

| \# | script/folder                                              | description                                           |
| :- | :--------------------------------------------------------- | :---------------------------------------------------- |
| 1  | [ğŸ“`datasets`](01-datasets)                                 | Dataset processing and characterisation               |
| 2  | [ğŸ“`metrics`](02-metrics)                                   | Metrics for comparing two trajectories                |
| 3  | [ğŸ“`methods`](03-methods)                                   | Trajectory inference methods                          |
| 4  | [ğŸ“`method_testing`](04-method_testing)                     | Method testing                                        |
| 5  | [ğŸ“`scaling`](05-scaling)                                   | Scalability of methods                                |
| 6  | [ğŸ“`benchmark`](06-benchmark)                               | Benchmarking of TI methods on real and synthetic data |
| 7  | [ğŸ“`stability`](07-stability)                               | Stability                                             |
| 8  | [ğŸ“`summary`](08-summary)                                   | Summarising the results into funky heatmaps           |
| 9  | [ğŸ“`guidelines`](09-guidelines)                             | Guidelines                                            |
| 10 | [ğŸ“`benchmark_interpretation`](10-benchmark_interpretation) | Benchmark interpretation                              |
| 11 | [ğŸ“`example_predictions`](11-example_predictions)           | Example predictions                                   |
| 12 | [ğŸ“`manuscript`](12-manuscript)                             | Preparing the manuscript                              |
|    | [ğŸ“`varia`](varia)                                          | Varia scripts                                         |
